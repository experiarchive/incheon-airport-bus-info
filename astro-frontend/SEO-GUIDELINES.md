## 2. 키워드 구현 규칙

### 2.1 키워드 밀도 가이드라인
- **주요 키워드 밀도**: 1.0-2.0% (권장: 1.5%)
- **보조 키워드 밀도**: 0.5-1.0%
- **롱테일 키워드**: 자연스럽게 산발적 배치

### 2.2 키워드 배치 우선순위

#### 1순위 배치 위치
1. **페이지 제목 (Title Tag)**: 주요 키워드 포함 필수
2. **메타 설명 (Meta Description)**: 주요 + 보조 키워드
3. **H1 태그**: 주요 키워드 포함
4. **첫 번째 단락**: 주요 키워드를 첫 100자 내 배치

#### 2순위 배치 위치
1. **H2, H3 태그**: 보조 키워드 자연스럽게 배치
2. **이미지 Alt 텍스트**: 관련 키워드 포함
3. **URL 슬러그**: 주요 키워드의 영문 번역 또는 로마자 표기

### 2.3 자연스러운 키워드 사용 규칙

#### DO (권장사항)
```
✅ "6001 공항버스 시간표를 확인하고 예매하세요"
✅ "인천공항 공항버스 요금 및 소요시간 안내"
✅ "공항버스 정류장 위치와 실시간 운행 정보"
```

#### DON'T (지양사항)
```
❌ "공항버스 공항버스 시간표 공항버스 예매"
❌ "공항버스공항버스공항버스" (키워드 스터핑)
❌ "airport bus 공항버스" (불필요한 영문 혼용)
```

## 3. 페이지별 키워드 전략

### 3.1 홈페이지 (index.astro)

#### 타겟 키워드 조합
**주요**: 공항버스, 공항버스 시간표, 공항버스 예매
**보조**: 인천공항, 공항버스 요금, 공항버스 정류장

#### 콘텐츠 키워드 전략
```html
<!-- H1 예시 -->
<h1>인천공항 공항버스 시간표 및 예매 - 실시간 운행정보</h1>

<!-- 첫 번째 단락 예시 -->
<p>인천공항 공항버스 시간표를 한눈에 확인하고 온라인 예매까지! 
전국 주요 지역에서 인천공항을 연결하는 공항버스 요금, 소요시간, 
정류장 위치를 실시간으로 제공합니다.</p>
```

#### 인기 노선 섹션 키워드
- "6001 공항버스 (강남↔인천공항)"

- "6009 공항버스 (김포공항↔인천공항)"

### 3.2 전체 노선 페이지 (routes.astro)

#### 타겟 키워드 조합
**주요**: [노선번호] 공항버스, 공항버스 노선, 공항버스 시간표

## 4. 콘텐츠 키워드 요구사항

### 4.1 필수 포함 키워드 (모든 페이지)
각 페이지는 다음 키워드들을 자연스럽게 포함해야 함:

1. **시간표** - 운행 스케줄 관련 섹션
2. **예매** - 예약/구매 관련 정보
3. **요금** - 가격 정보 섹션
4. **정류장** - 승차 위치 정보
5. **소요시간** - 이동 시간 정보

### 4.2 FAQ 섹션 키워드 전략

#### 자주 묻는 질문 키워드 활용
```markdown
## 자주 묻는 질문 (FAQ)

### Q: 공항버스 예매는 어떻게 하나요?
A: 공항버스 예매는 온라인과 현장에서 모두 가능합니다...

### Q: 공항버스 캐리어 반입 가능한가요?
A: 네, 공항버스에서는 캐리어 반입이 가능합니다...

### Q: 공항버스 요금은 얼마인가요?
A: 공항버스 요금은 노선별로 상이하며...

### Q: 공항버스 현금결제 가능한가요?
A: 공항버스는 현금결제와 카드결제 모두 지원합니다...
```

### 4.3 이미지 Alt 텍스트 최적화

#### 권장 Alt 텍스트 패턴
