# 제품 컨텍스트: 인천공항 공항버스 정보 웹사이트

## 프로젝트 존재 이유

### 해결하고자 하는 문제
1. **정보 분산 문제**
   - 공항버스 정보가 여러 웹사이트에 흩어져 있음
   - 사용자가 원하는 정보를 찾기 위해 여러 사이트를 방문해야 함

2. **실시간 정보 부족**
   - 정적인 시간표 정보만 제공되는 경우가 많음
   - 실시간 버스 위치 정보에 대한 접근성 부족

3. **모바일 최적화 부족**
   - 공항에서 모바일로 정보를 찾는 사용자가 많음
   - 기존 사이트들의 모바일 사용성 부족

4. **정류장 위치 정보 부족**
   - 공항 내 버스 정류장 위치를 찾기 어려움
   - 위치 정보가 부정확하거나 오래됨

## 사용자 경험 목표

### 핵심 사용자 여정

#### 1. 여행 계획 단계
**사용자 목표:** 공항버스 이용 가능성 확인 및 계획 수립
- **접근:** 홈페이지 → 전체 노선 보기
- **기대:** 목적지별 노선 쉽게 찾기
- **제공:** 깔끔한 노선 목록, 주요 경유지 정보

#### 2. 상세 정보 확인 단계
**사용자 목표:** 구체적인 시간표 및 정류장 정보 확인
- **접근:** 노선 목록 → 특정 노선 상세 페이지
- **기대:** 명확한 시간표, 정류장 위치 정보
- **제공:** 방향별/요일별 시간표, 정류장 위치 데이터

#### 3. 실시간 확인 단계
**사용자 목표:** 현재 버스 위치 및 도착 예정 시간 확인
- **접근:** 노선 상세 페이지 → 실시간 위치 확인
- **기대:** 실시간 버스 위치, 도착 예정 시간
- **제공:** KakaoMap iframe을 통한 실시간 정보

#### 4. 현장 이용 단계
**사용자 목표:** 공항에서 정류장 찾기 및 버스 탑승
- **접근:** 모바일에서 정류장 위치 확인
- **기대:** 정확한 정류장 위치, 찾기 쉬운 안내
- **제공:** 상세한 정류장 위치 정보, 공항 내 안내

### 사용자 인터페이스 원칙

#### 1. 직관성 (Intuitive)
- 첫 방문자도 쉽게 원하는 정보를 찾을 수 있어야 함
- 메뉴 구조가 논리적이고 예측 가능해야 함
- 버튼과 링크의 목적이 명확해야 함

#### 2. 속도 (Speed)
- 페이지 로딩 시간 3초 이내
- 필요한 정보에 3클릭 이내 접근
- Astro의 정적 생성 활용으로 빠른 응답

#### 3. 접근성 (Accessibility)
- 모든 디바이스에서 동일한 사용자 경험
- 시각적 계층 구조가 명확함
- 읽기 쉬운 폰트와 충분한 대비

#### 4. 신뢰성 (Reliability)
- 정확하고 최신의 정보 제공
- 일관된 디자인과 동작
- 오류 상황에 대한 명확한 안내

## 콘텐츠 전략

### 정보 우선순위
1. **1차 정보 (핵심)**
   - 버스 노선 번호 및 이름
   - 시간표 (출발 시간)
   - 주요 경유지

2. **2차 정보 (상세)**
   - 정류장 위치
   - 실시간 버스 위치
   - 요금 정보

3. **3차 정보 (보조)**
   - 공항 내 편의시설 안내
   - 무인 발권기 사용법
   - 기타 공항 이용 팁

### 콘텐츠 업데이트 전략
- **정적 콘텐츠:** Sanity.io CMS를 통한 관리
- **시간표:** 정기적 업데이트 (운수회사 공지 기반)
- **실시간 정보:** 외부 API/iframe 활용
- **일반 정보:** 필요시 수동 업데이트

## 수익화 모델

### AdSense 통합 계획
1. **위치 전략**
   - 헤더/푸터 배너
   - 콘텐츠 중간 삽입형 광고
   - 사이드바 (데스크톱)

2. **사용자 경험 고려**
   - 콘텐츠 가독성 방해하지 않는 위치
   - 모바일에서 터치 실수 방지
   - 로딩 속도에 영향 최소화

3. **콘텐츠 최적화**
   - 여행, 교통 관련 키워드 최적화
   - SEO 친화적 구조로 트래픽 증대
   - 정기적 콘텐츠 업데이트로 재방문 유도

## 성공 정의

### 단기 목표 (3개월)
- 기본 기능 완성 및 배포
- 주요 검색어 상위 10위 진입
- 일 방문자 100명 이상

### 중기 목표 (6개월)
- AdSense 승인 및 수익 창출 시작
- 월 방문자 10,000명 이상
- 사용자 피드백 기반 기능 개선

### 장기 목표 (1년)
- 월 광고 수익 $100 이상
- 공항버스 정보 검색 시 1위 사이트
- API 연동을 통한 실시간 정보 고도화
