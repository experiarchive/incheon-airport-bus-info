# 프로젝트 브리프: 인천공항 공항버스 정보 웹사이트

## 프로젝트 개요

**프로젝트명:** 인천공항 공항버스 정보 웹사이트  
**목표:** 사용자가 인천공항을 오가는 공항버스의 노선, 시간표, 정류장 위치, 실시간 버스 위치 및 기타 유용한 공항 정보를 쉽게 찾아볼 수 있는 빠르고 SEO 친화적인 웹사이트 구축  
**최종 목표:** Google AdSense를 통한 광고 수익 창출

## 핵심 요구사항

### 기능적 요구사항
1. **버스 노선 정보 표시**
   - 전체 버스 노선 목록
   - 노선별 상세 정보 (노선명, 번호, 설명, 주요 경유지)
   - 노선별 시간표 (공항행/시내행, 평일/주말/공휴일 구분)

2. **정류장 위치 정보**
   - JSON 데이터 기반 정류장 위치 표시
   - 위도/경도 정보 활용

3. **실시간 버스 위치**
   - 초기: KakaoMap iframe 연동
   - 향후: 공공데이터포털 API 연동 고려

4. **일반 정보 페이지**
   - 공항 내 정보 (버스 정류장 위치 안내)
   - 무인 발권기 사용 방법
   - 기타 공항 이용 가이드

### 기술적 요구사항
1. **빠른 로딩 속도** - Astro 정적 사이트 생성 활용
2. **SEO 최적화** - 검색 엔진 친화적 구조
3. **반응형 디자인** - 모바일, 태블릿, 데스크톱 최적화
4. **AdSense 연동 준비** - 광고 수익 창출 기반 마련

## 기술 스택

- **Frontend Framework:** Astro
- **Content Management:** Sanity.io (무료 플랜)
- **Styling:** Tailwind CSS
- **Hosting:** Vercel (무료 티어)
- **Data Sources:** JSON 파일 (정류장 위치), Sanity.io (콘텐츠)
- **Real-time Data:** KakaoMap iframe (초기), 공공데이터포털 API (향후)

## 사용자 프로필

**주요 사용자:** 인천공항을 이용하는 여행객
- 공항버스 이용을 계획하는 사람들
- 실시간 버스 위치를 확인하고 싶은 사람들
- 공항 내 버스 정류장 위치를 찾는 사람들
- 시간표를 확인하고 싶은 사람들

**사용 시나리오:**
1. 여행 전 버스 노선 및 시간표 확인
2. 공항에서 실시간 버스 위치 확인
3. 정류장 위치 찾기
4. 공항 내 편의시설 정보 확인

## 성공 지표

1. **사용자 경험**
   - 빠른 페이지 로딩 (3초 이내)
   - 직관적인 정보 접근
   - 모바일 친화적 인터페이스

2. **SEO 성과**
   - 검색 엔진 상위 노출
   - 유기적 트래픽 증가

3. **수익화**
   - AdSense 승인 및 광고 수익 창출

## 제약사항

- **예산:** 무료 서비스 활용 (Vercel, Sanity.io 무료 플랜)
- **개발자:** 비개발자 (AI 도움을 받는 셀프 개발)
- **데이터:** 공공 데이터 및 제공된 JSON 파일 활용
- **실시간 데이터:** 초기에는 iframe 기반, API는 향후 고려
